安装说明

  1. 下载 APK 文件到您的 Android 设备
  2. 在文件管理器中找到下载的文件并点击安装
  3. 如果提示"允许安装未知应用"，请开启此权限
  4. 安装完成后，按 Home 键或切换应用
  5. 选择 AI手机助手 并设为始终使用
  6. 首次打开会自动启动豆包语音助手

⚙️ 安装后设置

1 设置默认启动器（开机自动启动）

让 AI手机助手 在开机时自动启动：

  1. 按设备的 Home 键 或 最近应用键
  2. 在弹出的选择器中，选择 AI手机助手
  3. 点击 "始终" 或 "设为默认"
  4. 以后开机或按 Home 键都会进入 AI手机助手

✅ 完成后，开机将自动进入 AI手机助手 并启动豆包语音助手

2 启用无障碍服务（应用锁定功能）

防止意外退出豆包语音助手：

  1. 打开手机 设置
  2. 找到 无障碍 或 辅助功能
  3. 向下滚动找到 AI手机助手
  4. 开启开关，授予无障碍权限

✅ 启用后，在豆包应用中按返回键将无法退出，实现真正的锁定

3 了解解锁方式

当需要退出豆包或卸载应用时，使用以下方法解锁：

方法一：音量键解锁

  • 同时按下 音量+ 和 音量-
  • 或交替按 音量+、音量-、音量+ 三次

方法二：触摸解锁

  • 在 AI手机助手 桌面连续点击 5 次

⚠️ 解锁后会提示"已解锁"，此时可以正常使用其他应用或卸载本应用

4 还原默认启动器

如需恢复原来的桌面：

  1. 先解锁应用（见步骤3）
  2. 打开手机 设置
  3. 找到 应用管理 或 应用
  4. 找到 AI手机助手
  5. 点击 清除默认设置 或 默认应用
  6. 按 Home 键重新选择原来的桌面
